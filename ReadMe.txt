### Requirements

- A code editor (VS Code, Sublime Text, etc.)
- A modern web browser (Chrome, Firefox, etc.)

### Steps to follow

1. **Download or clone the project:**

2. **Click the "Test workflow" button to launch the rendering in the environment.**
